# Agent Guidelines for PlayCanvas Blog

## Project Overview

Official PlayCanvas blog built with Docusaurus 3.

- **Language**: Markdown
- **Framework**: Docusaurus 3
- **License**: MIT

## Directory Structure

```
blog/       # Blog posts (YYYY-MM-DD-slug.md format)
static/img/ # Images for posts
```

## Writing Style

- Use American English
- No Oxford commas
- Emojis are encouraged to keep posts fun and engaging

## Commands

```bash
npm run start  # Development server
npm run build  # Production build
npm run lint   # Lint Markdown files
```
